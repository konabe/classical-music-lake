# ADR-001: AWS WAF を導入しない

- **ステータス**: 決定済み
- **日付**: 2026-04-03
- **関連Issue**: #291

## コンテキスト

API Gateway および CloudFront に対する攻撃防御のため、AWS WAF の導入を検討した（Issue #291）。

AWS WAF の料金は以下の通り。

| 項目       | 料金                  |
| ---------- | --------------------- |
| Web ACL    | $5.00/月              |
| ルール     | $1.00/月/ルール       |
| リクエスト | $0.60/100万リクエスト |

## 決定

**AWS WAF は導入しない。**

## 理由

本サービスは個人利用を目的とした非収益プロジェクトであり、WAF の月額コスト（最低でも $6〜10/月）を継続的に負担することが困難なため。

## トレードオフ

- **デメリット**: SQLインジェクション・XSS・DDoS などの攻撃に対する防御層が存在しない
- **代替対策**: Lambda 内の Zod バリデーション、Cognito Authorizer による認証済みリクエストのみの受け付け、CloudFront によるエッジでの基本的なフィルタリングで対応する

## 再検討条件

サービスが収益化された場合、または深刻なセキュリティインシデントが発生した場合に再検討する。
