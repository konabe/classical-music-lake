# 楽曲マスタ機能 要件定義

## 概要

クラシック音楽の楽曲をマスタデータとして登録・管理する機能。
鑑賞ログ入力時に登録済み楽曲を選択できる補助機能も提供する。

---

## 機能要件

### 1. 楽曲の登録

- 曲名・作曲家を入力して楽曲を新規登録できる
- 必須項目: 曲名、作曲家

### 2. 楽曲の一覧表示

- 登録済みの楽曲を一覧で確認できる
- 表示項目: 曲名、作曲家

### 3. 楽曲の編集

- 登録済みの楽曲の曲名・作曲家を編集できる

### 4. 楽曲の削除

- 登録済みの楽曲を削除できる

### 5. 鑑賞ログ入力時の補助

- 鑑賞ログの新規作成・編集フォームで、登録済み楽曲から選択できる
- 楽曲を選択すると曲名・作曲家が自動入力される

---

## データモデル

### Piece（楽曲）

| フィールド  | 型     | 必須 | 説明                 |
| ----------- | ------ | ---- | -------------------- |
| `id`        | string | -    | UUID（自動生成）     |
| `title`     | string | ✅   | 曲名                 |
| `composer`  | string | ✅   | 作曲家名             |
| `createdAt` | string | -    | 作成日時（ISO 8601） |
| `updatedAt` | string | -    | 更新日時（ISO 8601） |

---

## API

| メソッド | パス         | 説明         |
| -------- | ------------ | ------------ |
| GET      | /pieces      | 楽曲一覧取得 |
| POST     | /pieces      | 楽曲新規登録 |
| PUT      | /pieces/{id} | 楽曲更新     |
| DELETE   | /pieces/{id} | 楽曲削除     |

---

## UI

| ページ   | パス              | 説明                     |
| -------- | ----------------- | ------------------------ |
| 楽曲一覧 | /pieces           | 登録済み楽曲の一覧・削除 |
| 楽曲登録 | /pieces/new       | 新規登録フォーム         |
| 楽曲編集 | /pieces/{id}/edit | 編集フォーム             |

---

## 非機能要件

- 既存の鑑賞ログ機能との一貫したUI/UXを維持する
- 楽曲の個別詳細ページは作成しない（一覧から直接編集・削除）
