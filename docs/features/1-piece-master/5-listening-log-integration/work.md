# ワーク5: 鑑賞ログ連携

## 概要

鑑賞ログの新規作成・編集フォームに、登録済み楽曲から選択して曲名・作曲家を自動入力する補助機能を追加する。

## 実装内容

### フロントエンド

- `components/ListeningLogForm.vue` を更新
  - 楽曲選択 UI（セレクトボックスまたは検索）を追加
  - 楽曲を選択すると `composer`・`piece`（曲名）フィールドに自動入力される
  - 選択せず手動入力することも引き続き可能
- `GET /pieces` を呼び出す composable を追加（`composables/usePieces.ts`）

## 動作確認

- 鑑賞ログフォームで楽曲を選択すると曲名・作曲家が自動入力されること
- 楽曲を選択せずに手動で入力・保存できること
