# ワーク2: 楽曲カテゴリの表示対応

## ユーザーストーリー

楽曲一覧画面・詳細画面でカテゴリ情報を確認できるようにする。

## 要件

- 楽曲一覧画面で各楽曲のカテゴリ情報を表示する
- 楽曲詳細画面でカテゴリ情報を表示する
- カテゴリが未設定の場合は表示しない（空欄やプレースホルダーを出さない）
- 表示はラベル形式で、どの軸のカテゴリかが分かるようにする

## 受入テストケース

- [ ] 楽曲一覧画面で、カテゴリが設定されている楽曲にカテゴリ情報が表示される
- [ ] 楽曲一覧画面で、カテゴリが未設定の楽曲には何も表示されない
- [ ] 楽曲詳細画面で、設定されたカテゴリ情報が表示される
- [ ] 楽曲詳細画面で、カテゴリが未設定の場合は何も表示されない
- [ ] 一部のカテゴリのみ設定されている場合、設定されたカテゴリのみ表示される
- [ ] 4軸すべて設定されている場合、すべてのカテゴリが正しく表示される
