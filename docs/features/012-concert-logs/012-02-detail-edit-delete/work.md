# ワーク012-02: コンサート記録の詳細・編集・削除

## ユーザーストーリー

ユーザーとして、記録したコンサートの詳細を閲覧し、内容を編集または削除できる。

## 要件

### 機能要件

- コンサート記録の詳細画面で、記録した内容を確認できる
- コンサート記録の編集画面で、既存の記録内容を変更できる
- コンサート記録を削除できる
- ログインユーザーのみがアクセスできる（認証必須）
- 他のユーザーの記録にはアクセスできない（存在を隠蔽して 404 を返す）

### 詳細画面

- 記録されたすべてのフィールドを表示する
- 編集画面への導線を設ける
- 削除操作が可能

### 編集画面

- 既存の記録内容がフォームに初期表示される
- 部分更新（変更したフィールドのみ送信）をサポートする
- 更新後、詳細画面に遷移する

### 削除

- 削除前に確認を求める
- 削除後、一覧画面に遷移する

## 受入テストケース

### 詳細

- [ ] 一覧画面からコンサート記録をクリックして詳細画面に遷移できる
- [ ] 詳細画面に日時・会場・指揮者・オーケストラ・ソリストが表示される
- [ ] 任意項目が未入力の場合、その項目は表示されない（または空欄で表示）
- [ ] 他ユーザーの記録の詳細画面にアクセスすると 404 が表示される

### 編集

- [ ] 詳細画面から編集画面に遷移できる
- [ ] 編集画面に既存の記録内容が初期表示される
- [ ] フィールドを変更して保存できる
- [ ] 会場を空にするとバリデーションエラーになる
- [ ] 更新後、詳細画面に遷移する
- [ ] 他ユーザーの記録を編集しようとすると 404 が表示される

### 削除

- [ ] 詳細画面から削除操作ができる
- [ ] 削除前に確認ダイアログが表示される
- [ ] 削除後、一覧画面に遷移する
- [ ] 他ユーザーの記録を削除しようとすると 404 が表示される

### 認証

- [ ] 未ログイン状態で詳細・編集画面にアクセスすると、ログイン画面にリダイレクトされる
