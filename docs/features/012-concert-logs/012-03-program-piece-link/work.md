# ワーク012-03: プログラム（楽曲マスタ連携）

## ユーザーストーリー

ユーザーとして、コンサート記録にプログラム（演奏曲目）を楽曲マスタから複数選択して紐付けできる。

## 要件

### 機能要件

- コンサート記録の作成・編集時に、楽曲マスタ（Piece）から複数の曲を選択してプログラムとして紐付けできる
- コンサート記録の詳細画面で、プログラム（紐付けられた楽曲）の一覧が表示される
- プログラムの表示順序を保持できる（演奏順）
- 楽曲マスタに登録されていない曲は選択できない

### 作成・編集画面

- 楽曲マスタから曲を検索・選択して追加できる
- 複数の曲を追加できる
- 追加した曲の並び順を変更できる（演奏順の管理）
- 追加した曲を削除できる

### 詳細画面

- プログラムとして紐付けられた楽曲の一覧が演奏順で表示される
- 各楽曲の曲名・作曲家が表示される

## 受入テストケース

### 作成時のプログラム設定

- [ ] コンサート記録の作成画面で、楽曲マスタから曲を選択して追加できる
- [ ] 複数の曲を追加できる
- [ ] 追加した曲の並び順を変更できる
- [ ] 追加した曲を個別に削除できる
- [ ] プログラムなし（0曲）でもコンサート記録を作成できる
- [ ] 作成後、詳細画面でプログラムが正しく表示される

### 編集時のプログラム変更

- [ ] 編集画面で既存のプログラムが初期表示される
- [ ] 曲を追加・削除・並べ替えして更新できる
- [ ] プログラムをすべて削除して更新できる

### 詳細画面でのプログラム表示

- [ ] 詳細画面にプログラム（楽曲一覧）が演奏順で表示される
- [ ] 各楽曲の曲名・作曲家が表示される
- [ ] プログラムが未設定の場合、適切なメッセージが表示される
