# ワーク: TOPページへの管理者向けリンクセクション追加

## 概要

TOPページ（`pages/index.vue`）の下部に、管理者向けリンクセクションを追加する。
初期リンクとして楽曲マスタ（`/pieces`）へのテキストリンクを配置する。

## 要件

- TOPページのメインコンテンツ（カードセクション）の下部に、管理者向けリンクをまとめたセクションを設ける
- 楽曲マスタ（`/pieces`）へのテキストリンクをそのセクション内に配置する
- リンクは「管理者向け」の位置づけのため、メインカードより視覚的に目立たない控えめなデザインにする
- 既存の「鑑賞記録」カードのデザイン・レイアウトに影響を与えないこと
- 将来的に管理者向けリンクが増えた際に追加しやすい構造にすること

## テストケース

| #   | 操作                               | 期待結果                                     |
| --- | ---------------------------------- | -------------------------------------------- |
| 1   | TOPページを開く                    | 管理者向けリンクセクションが表示される       |
| 2   | TOPページを開く                    | 「楽曲マスタ」のテキストリンクが表示される   |
| 3   | 「楽曲マスタ」リンクをクリックする | `/pieces` に遷移する                         |
| 4   | TOPページを開く                    | 「鑑賞記録」カードが引き続き正常に表示される |
