# 005-001 ヘッダー未ログイン時の導線追加

## 概要

未ログイン状態のユーザーがどのページからでも「新規登録」「ログイン」ページへアクセスできるよう、グローバルヘッダーにナビゲーションリンクを追加する。

## ユーザーストーリー

未ログインのユーザーとして、ヘッダーから新規登録・ログインページへ移動したい。
なぜなら、どのページを見ていても認証操作へ素早くアクセスしたいから。

## 受入テストケース

- [ ] 未ログイン状態でどのページを表示しても、ヘッダーに「新規登録」リンクが表示される
- [ ] 未ログイン状態でどのページを表示しても、ヘッダーに「ログイン」リンクが表示される
- [ ] 「新規登録」リンクをクリックすると、新規登録ページへ遷移する
- [ ] 「ログイン」リンクをクリックすると、ログインページへ遷移する
