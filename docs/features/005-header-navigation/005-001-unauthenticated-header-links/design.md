# 005-001 ヘッダー未ログイン時の導線追加 - 設計

## 変更対象

`app/layouts/default.vue`

## 設計概要

既存の `default.vue` にはすでに以下が実装されている：

- `useAuth()` から `isAuthenticated` を取得し、`isLoggedIn` としてリアクティブに保持
- ルート変化を `watch` で監視し `isLoggedIn` を更新
- `v-if="isLoggedIn"` によるログアウトボタンの表示制御

この既存の仕組みを活用し、未ログイン時に「新規登録」「ログイン」リンクを表示するナビゲーション要素を追加する。

## UI 変更

### 追加要素

| 要素           | 表示条件             | 遷移先                |
| -------------- | -------------------- | --------------------- |
| 新規登録リンク | `!isLoggedIn` のとき | `/auth/user-register` |
| ログインリンク | `!isLoggedIn` のとき | `/auth/login`         |

### 配置

ヘッダー右端エリアに既存のログアウトボタンと同じ位置に配置する。
未ログイン時は「新規登録」「ログイン」リンクを表示し、ログイン済み時はログアウトボタンのみを表示する。

### スタイル

既存の `.logout-button` クラスに準じたスタイルを適用する。
リンクはボタン形式ではなくテキストリンクで構いない（ログアウトボタンとの視覚的区別のため）。

## 認証状態の判定

既存の `isLoggedIn` リアクティブ変数（`ref(isAuthenticated())` + `watch` による更新）をそのまま利用する。追加の実装は不要。
