# 005-002 ヘッダーログイン済み時のログアウト導線追加 - 設計

## 変更対象

`app/layouts/default.vue`

## 設計概要

既存の `default.vue` にはすでに以下が実装されている：

- `useAuth()` から `isAuthenticated` と `logout` を取得
- `isLoggedIn` リアクティブ変数（`ref(isAuthenticated())` + ルート変化を `watch` で監視）
- `v-if="isLoggedIn"` によるログアウトボタンの表示（テンプレート実装済み）

現時点でログアウトボタン自体は実装済みのため、本ワークでは **005-001 と同じ変更（未ログイン時の「新規登録」「ログイン」リンク追加）によって、ログイン済み時にそれらが表示されないことが自動的に保証される**。

## UI 変更

### ログアウトボタン（既存実装）

| 要素             | 表示条件            | 動作                                   |
| ---------------- | ------------------- | -------------------------------------- |
| ログアウトボタン | `isLoggedIn` のとき | `logout()` 実行後 `/auth/login` へ遷移 |

### 「新規登録」「ログイン」の非表示（005-001 との連携）

005-001 で `v-if="!isLoggedIn"` を付与することにより、ログイン済み時は自動的に非表示となる。本ワーク単体での追加実装はない。

## 実装方針

005-001 と 005-002 はどちらも `app/layouts/default.vue` の同一箇所を変更するため、**1 つの実装 PR でまとめて対応することを推奨する**。

- 005-001: `v-if="!isLoggedIn"` で「新規登録」「ログイン」リンクを追加
- 005-002: `v-if="isLoggedIn"` のログアウトボタンは既存実装済み（追加作業なし）
