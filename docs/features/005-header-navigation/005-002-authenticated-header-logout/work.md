# 005-002 ヘッダーログイン済み時のログアウト導線追加

## 概要

ログイン済みユーザーがどのページからでもログアウトできるよう、グローバルヘッダーに「ログアウト」ボタンを追加する。また、ログイン済み時は「新規登録」「ログイン」リンクを非表示にする。

## ユーザーストーリー

ログイン済みのユーザーとして、ヘッダーからログアウトしたい。
なぜなら、どのページを操作中でも素早くログアウトできるようにしたいから。

## 受入テストケース

- [ ] ログイン済み状態でどのページを表示しても、ヘッダーに「ログアウト」ボタンが表示される
- [ ] ログイン済み状態では、ヘッダーに「新規登録」「ログイン」リンクが表示されない
- [ ] 「ログアウト」ボタンをクリックすると、ログアウトされログインページへ遷移する
- [ ] 未ログイン状態では、ヘッダーに「ログアウト」ボタンが表示されない
