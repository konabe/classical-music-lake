# ヘッダーナビゲーション - 要件定義

## 概要

グローバルヘッダーにナビゲーションリンクを追加し、未ログイン時は新規登録・ログインへの導線を、ログイン済み時はログアウトボタンを表示する。

## 目的

ユーザーがどのページからでも認証関連の操作（新規登録・ログイン・ログアウト）へ素早くアクセスできるようにする。

## 機能スコープ

### 実装する機能

- ヘッダーに認証状態に応じたナビゲーションを追加する
  - 未ログイン時: 「新規登録」「ログイン」へのリンクを表示
  - ログイン済み時: 「ログアウト」ボタンを表示

### 実装しない機能（将来の実装対象）

- ヘッダーへのユーザー名・アバター表示
- ドロップダウンメニュー

## 設計申し送りメモ

- 認証状態の判定は既存の認証ロジック（Cognito）を活用すること
- ヘッダーは既存の `app/layouts/default.vue` に実装すること
