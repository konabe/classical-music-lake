# 楽曲詳細ページで動画を視聴しながらログを記録できる - ワーク定義

## ユーザーストーリー

クラシック音楽を探索したいユーザーとして、楽曲詳細ページで動画を視聴しながら、
その場でメモを書いていつでも鑑賞ログとして保存できる。
これにより、「曲を探す → 聴く → 記録する」という体験がシームレスにつながる。

## 機能概要

- 楽曲マスタ一覧（`/pieces`）の各アイテムから詳細ページ（`/pieces/[id]`）へ遷移できる
- 詳細ページには楽曲情報（曲名・作曲家）と、`videoUrl` が設定されている場合は動画プレイヤーを表示する
- 動画の再生を開始した時点で、クイックログフォームが表示される
  - 作曲家・曲名は Piece の情報から自動入力済み
  - メモ・評価・お気に入りを入力できる
- 「記録する」ボタンを押すと視聴ログ（`ListeningLog`）として保存される
  - 視聴日時（`listenedAt`）は保存時点の日時を使用する
- `videoUrl` が未設定の楽曲でも詳細ページは表示される（動画・クイックログフォームは非表示）

## 受入テストケース

### 楽曲一覧からの導線

- [ ] 楽曲一覧の各アイテムに詳細ページへのリンクが表示されている
- [ ] リンクをクリックすると `/pieces/[id]` に遷移する

### 楽曲詳細ページ（videoUrl あり）

- [ ] 曲名・作曲家が表示されている
- [ ] YouTube 動画プレイヤーが表示されている
- [ ] 動画再生前はクイックログフォームが表示されていない
- [ ] 動画を再生するとクイックログフォームが表示される
- [ ] クイックログフォームに作曲家・曲名が自動入力されている
- [ ] メモ・評価・お気に入りを入力できる
- [ ] 「記録する」を押すと視聴ログが保存され、完了メッセージが表示される
- [ ] 保存後もページに留まり、再度記録できる状態になる

### 楽曲詳細ページ（videoUrl なし）

- [ ] 曲名・作曲家が表示されている
- [ ] 動画プレイヤーが表示されない
- [ ] クイックログフォームが表示されない
