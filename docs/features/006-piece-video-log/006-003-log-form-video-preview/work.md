# 視聴ログ作成フォームで楽曲の動画を確認できる - ワーク定義

## ユーザーストーリー

視聴ログを記録するユーザーとして、楽曲マスタから曲を選択したときに動画が表示される。
これにより、記録しようとしている曲が正しいか動画で確認しながらログを入力できる。

## 機能概要

- 視聴ログ新規作成・編集フォームの楽曲マスタ選択で曲を選んだ際、
  選択した Piece に `videoUrl` が設定されていれば動画プレイヤーをインライン表示する
- 別の曲を選択した場合、動画は切り替わる
- 「選択しない」を選んだ場合、動画は非表示になる
- `videoUrl` が未設定の曲を選択した場合、動画は表示されない

## 受入テストケース

### 動画あり楽曲を選択した場合

- [ ] 楽曲マスタのセレクトで `videoUrl` ありの曲を選択すると動画プレイヤーが表示される
- [ ] 表示された動画は選択した曲のものである
- [ ] 動画を見ながらフォームの他の項目（評価・メモ等）を入力できる

### 動画なし楽曲を選択した場合

- [ ] `videoUrl` なしの曲を選択しても動画プレイヤーは表示されない

### 選択を変更した場合

- [ ] 別の曲（`videoUrl` あり）に選択を変えると動画が切り替わる
- [ ] 「選択しない」にすると動画が非表示になる
