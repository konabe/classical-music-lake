# 楽曲マスタに動画URLを登録・編集できる - ワーク定義

## ユーザーストーリー

楽曲マスタを管理するユーザーとして、楽曲に動画 URL を登録・編集できる。
これにより、楽曲と動画を紐づけるデータ基盤が整い、後続の動画視聴機能の土台となる。

## 機能概要

- 楽曲マスタ（Piece）に `videoUrl` フィールド（任意）を追加する
- 楽曲の新規作成フォームに動画 URL 入力欄を追加する
- 楽曲の編集フォームに動画 URL 入力欄を追加する
- 既存の楽曲は `videoUrl` が未設定の状態として扱う（後から編集で追加可能）

## 受入テストケース

### 楽曲新規作成

- [ ] 動画 URL 入力欄が表示されている
- [ ] 動画 URL を入力せずに楽曲を作成できる（任意項目）
- [ ] 有効な YouTube URL を入力して楽曲を作成できる
- [ ] 作成後、楽曲一覧に遷移する

### 楽曲編集

- [ ] 既存の楽曲（videoUrl なし）の編集フォームに動画 URL 入力欄が表示されている
- [ ] 動画 URL を追加して保存できる
- [ ] 登録済みの動画 URL が編集フォームに初期値として表示される
- [ ] 動画 URL を削除（空文字）して保存できる
