# 004-002 登録後の認証コード入力画面への自動遷移

## 概要

ユーザー登録が完了した直後に、自動的に認証コード入力画面へ遷移する。

## ユーザーストーリー

> ユーザーとして、登録完了後にすぐ認証コード入力画面へ案内されることで、次に何をすればよいか迷わずにメール確認を完了したい。

## 機能詳細

- ユーザー登録フォームの送信が成功した後、認証コード入力画面へ自動的にリダイレクトする
- リダイレクト先の認証コード入力画面には、登録したメールアドレスが（マスク表示等で）示される

## 受入テストケース

- [ ] ユーザー登録フォームを正常に送信すると、認証コード入力画面へ遷移する
- [ ] 登録エラーが発生した場合は、登録画面にとどまりエラーが表示される（遷移しない）
- [ ] 認証コード入力画面に、登録したメールアドレスが表示される
