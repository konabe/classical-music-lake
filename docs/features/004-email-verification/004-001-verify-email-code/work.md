# 004-001 認証コード入力画面

## 概要

メールアドレス確認のための認証コードを入力・送信する画面を提供する。
コードが正しければ自動ログインし、誤りや期限切れの場合はエラーを表示する。
再送信ボタンから認証コードを再発行できる。

## ユーザーストーリー

> ユーザーとして、受信したメールの認証コードを入力して自分のメールアドレスを確認し、そのままサービスを使い始めたい。

## 機能詳細

- 認証コード入力フォームを持つ画面を用意する
- 「確認する」ボタンを押すと認証コードをバックエンドへ送信する
- 認証コードが正しい場合：ログイン状態にしてトップ画面（または適切な画面）へ遷移する
- 認証コードが無効・期限切れの場合：エラーメッセージを画面上に表示する
- 「再送信」ボタンを押すと認証コードの再発行をリクエストし、成功時は案内メッセージを表示する

## 受入テストケース

- [ ] 認証コード入力フォームが表示される
- [ ] 正しい認証コードを入力して送信すると、自動的にログイン状態になり画面遷移する
- [ ] 誤った認証コードを入力して送信すると、エラーメッセージが表示される
- [ ] 期限切れの認証コードを入力して送信すると、エラーメッセージが表示される
- [ ] 「再送信」ボタンを押すと認証コードの再送信が行われ、案内メッセージが表示される
- [ ] 再送信に失敗した場合、エラーメッセージが表示される
