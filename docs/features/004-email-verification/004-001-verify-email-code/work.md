# 004-001 認証コード入力画面・登録後の自動遷移

## 概要

ユーザー登録完了後に認証コード入力画面へ自動遷移し、メールで受け取った認証コードを入力・検証する。
コードが正しければ自動ログイン、誤りや期限切れの場合はエラーを表示し、再送信ボタンからコードを再発行できる。

## ユーザーストーリー

> ユーザーとして、登録完了後すぐに認証コード入力画面へ案内され、受信したコードを入力してそのままサービスを使い始めたい。

## 機能詳細

- ユーザー登録フォームの送信が成功した後、認証コード入力画面へ自動的にリダイレクトする
- 認証コード入力画面には登録したメールアドレスが表示される
- 認証コード入力フォームを持つ画面を用意する
- 「確認する」ボタンを押すと認証コードをバックエンドへ送信する
- 認証コードが正しい場合：ログイン状態にしてトップ画面（または適切な画面）へ遷移する
- 認証コードが無効・期限切れの場合：エラーメッセージを画面上に表示する
- 「再送信」ボタンを押すと認証コードの再発行をリクエストし、成功時は案内メッセージを表示する
- 再送信に失敗した場合はエラーメッセージを表示する

## 受入テストケース

- [x] ユーザー登録フォームを正常に送信すると、認証コード入力画面へ遷移する
- [x] 登録エラーが発生した場合は、登録画面にとどまりエラーが表示される（遷移しない）
- [x] 認証コード入力画面に、登録したメールアドレスが表示される
- [x] 認証コード入力フォームが表示される
- [x] 正しい認証コードを入力して送信すると、自動的にログイン状態になり画面遷移する
- [x] 誤った認証コードを入力して送信すると、エラーメッセージが表示される
- [x] 期限切れの認証コードを入力して送信すると、エラーメッセージが表示される
- [x] 「再送信」ボタンを押すと認証コードの再送信が行われ、案内メッセージが表示される
- [x] 再送信に失敗した場合、エラーメッセージが表示される
