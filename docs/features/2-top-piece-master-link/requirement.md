# 要件: TOP画面への楽曲マスタ導線追加

## 背景

TOP画面には現在「鑑賞記録」へのカードリンクのみ存在する。
楽曲マスタ（`/pieces`）への導線がなく、管理操作のたびに URL を直接入力する必要がある。

## 要件

### 機能要件

- TOP画面のページ下部に、楽曲マスタ（`/pieces`）へのテキストリンクを追加する
- リンクは「管理者向け」という位置づけのため、メインカードより目立たない控えめなデザインにする
- リンクをクリックすると `/pieces` に遷移する

### 非機能要件

- 既存の「鑑賞記録」カードの視認性・デザインに影響を与えないこと
- 将来的に管理者向けリンクが増えることを想定した配置にすること

## 対象画面

- `pages/index.vue`（TOPページ）
