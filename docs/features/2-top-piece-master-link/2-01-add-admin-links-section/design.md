# 設計: TOPページへの管理者向けリンクセクション追加

## 変更対象

- `pages/index.vue`

## 画面構成

TOPページの既存レイアウトに、以下の順でセクションを配置する。

```
[hero セクション]         ← 変更なし
[menu-cards セクション]   ← 変更なし（鑑賞記録カード）
[admin-links セクション]  ← 新規追加
```

## `admin-links` セクションの設計

### マークアップ構造

- `<section class="admin-links">` でセクションを定義する
- セクション内に `<NuxtLink to="/pieces">楽曲マスタ</NuxtLink>` を配置する
- 将来のリンク追加に対応できるよう、リンクを `<ul>` / `<li>` で列挙する構造にする

### スタイル方針

- フォントサイズ・カラーはページ内の補助テキスト（`.card p` の `color: #888`）に合わせ、控えめに表示する
- テキストリンクのみ（カードスタイルは使わない）
- ページ下部に margin-top で間隔を空けて配置する

## テスト設計

| テストケース                             | 検証方法                                   |
| ---------------------------------------- | ------------------------------------------ |
| 管理者向けリンクセクションが表示される   | `section.admin-links` の存在を確認         |
| 「楽曲マスタ」テキストリンクが表示される | `a[href="/pieces"]` の存在とテキストを確認 |
| 「鑑賞記録」カードが引き続き表示される   | `a[href="/listening-logs"]` の存在を確認   |
